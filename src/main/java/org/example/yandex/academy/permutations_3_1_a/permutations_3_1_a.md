# Перестановки

Выведите число перестановок P(n).

### Формат ввода
В первой строке находится одно число (1 ≤ n ≤ 7).

### Формат вывода
Выведите ответ на задачу.

### Пример 1
##### Ввод
    2
##### Вывод
    2

### Пример 2
##### Ввод
    1
##### Вывод
    1

### Пример 3
##### Ввод
    3
##### Вывод
    6

### Ограничение памяти
    256.0 Мб
### Ограничение времени
    1 с
### Ввод
    стандартный ввод или input.txt
### Вывод
    стандартный вывод или output.txt
